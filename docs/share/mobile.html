移动端

Tiger小于 1 分钟

  • 适配方案

    1. 安装依赖
    yarn add lib-flexible
yarn add postcss-pxtorem
    1. 在main.js入口文件引入
    import 'lib-flexible'
    1. 新增配置文件 postcss.config.js
    module.exports = {
  plugins: {
    'postcss-pxtorem': {
      rootValue ({ file }) {
        return file.indexOf('vant') !== -1 ? 37.5 : 75
      },
      propList: ['*']
    }
  }
}

    注意

    • 可能会有版本不兼容的报错
    • 可能需要安装postcss、postcss-loader解析器
    • less-loader与postcss-loader的版本要相互匹配，版本相差不能太大
